key Authentication

All API requests require an API key sent in the X-API-Key header.

POST /api/v1/auth/verify

Verify your API key

Example Request

curl -X GET https://api.allosint.dev/v1/auth/verify \
  -H "X-API-Key: your_api_key_here"

Response (200 OK)

{
  "status": "success",
  "data": {
    "valid": true,
    "rate_limit": 1000,
    "remaining": 876,
    "reset": 1620000000
  }
}

dns Subdomain Enumeration

Discover subdomains for a given domain name.

GET /api/v1/subdomains?domain=example.com

Get all discovered subdomains

Parameters

Parameter Type Required Description
domain string Yes The domain to search subdomains for
max_results integer No Maximum number of results to return (default: 100)

Example Request

curl -X GET \
  'https://api.allosint.dev/v1/subdomains?domain=example.com&max_results=50' \
  -H 'X-API-Key: your_api_key_here'

Response (200 OK)

{
  "status": "success",
  "data": {
    "domain": "example.com",
    "subdomains": [
      "www.example.com",
      "mail.example.com",
      "api.example.com",
      "dev.example.com"
    ],
    "count": 4,
    "timestamp": 1620000000
  }
}

image_search Reverse Image Search

Search the web for similar or matching images.

POST /api/v1/image/search

Search by image

Parameters

Parameter Type Required Description
image file Yes* Image file to search (either image or image_url required)
image_url string Yes* URL of the image to search (either image or image_url required)
limit integer No Maximum number of results to return (default: 10, max: 50)
sites array No Array of domains to limit the search to

Example Request

curl -X POST \
  'https://api.allosint.dev/v1/image/search' \
  -H 'X-API-Key: your_api_key_here' \
  -F 'image=@/path/to/image.jpg' \
  -F 'limit=5'
