info Introduction

Welcome to the AllOSINT documentation! This guide will help you get started with the AllOSINT platform, a powerful open-source intelligence (OSINT) tool designed for security researchers, penetration testers, and intelligence analysts.

What is AllOSINT?

AllOSINT is a comprehensive OSINT framework that automates the process of gathering intelligence from publicly available sources. It provides a unified interface to access multiple OSINT tools and data sources, making it easier to collect, analyze, and visualize intelligence data.

download Installation

AllOSINT can be installed on Windows, macOS, and Linux systems. The recommended way to install AllOSINT is using Docker for easy dependency management.

Prerequisites

  • Docker and Docker Compose
  • Python 3.8+ (for development)
  • Git
  • Minimum 4GB RAM (8GB recommended)

Docker Installation (Recommended)

# Clone the repository
git clone https://github.com/allosint/allosint.git
cd allosint

# Copy the example environment file
cp .env.example .env

# Build and start the containers
docker-compose up -d

# Access the web interface at http://localhost:8000

Manual Installation

# Clone the repository
git clone https://github.com/allosint/allosint.git
cd allosint

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set up the database
python manage.py migrate

# Create a superuser (follow prompts)
python manage.py createsuperuser

# Run the development server
python manage.py runserver
info

Note for Windows Users

Make sure to run the command prompt as Administrator when installing dependencies or running Docker commands. Some features might require additional configuration on Windows.

rocket_launch Quick Start

This quick start guide will walk you through your first OSINT scan with AllOSINT.

1. Logging In

After installation, access the web interface at http://localhost:8000 and log in with your credentials.

2. Create a New Project

  1. Click on New Project in the sidebar
  2. Enter a project name and description
  3. Select the type of intelligence you want to gather
  4. Click Create Project

3. Run Your First Scan

From your project dashboard, you can start different types of scans:

Domain Reconnaissance

Discover subdomains, DNS records, and associated IPs.

Email Breach Check

Check if an email appears in known data breaches.

4. View and Export Results

Once your scan is complete, you can:

  • View results in the interactive dashboard
  • Export data in multiple formats (JSON, CSV, PDF)
  • Generate reports with visualizations
  • Schedule recurring scans

Pro Tip

Use the Quick Scan feature for common OSINT tasks without creating a project. Access it from the top navigation bar.
